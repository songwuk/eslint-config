<h2>Usage</h2>

<h3>install</h3>

```bash
pnpm add eslint @songwuk/eslint-config-basic -D
```

<h3>Config</h3>

`.eslintrc`
```json
{
  "extends": "@songwuk/eslint-config-basic"
}
```
